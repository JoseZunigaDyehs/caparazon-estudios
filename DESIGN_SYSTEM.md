# Caparazón Estudios - Design System

## Brand Identity
**Name:** Caparazón Estudios
**Vision:** High-engineering, premium, ultra-minimalist.

## Technical Stack
- **Framework:** Astro 7
- **CSS:** Tailwind CSS 4.3
- **Syntax:** Pure .astro components, native Tailwind classes.

## Visual Style
- **Theme:** Native Dark Mode.
- **Atmosphere:** Ultra-minimalist, premium, high-engineering.
- **Layout:** Wide spacing (`tracking-wide`), mobile-first, semantic HTML.
- **Shapes:** Fine 1px borders, slightly rounded corners (`rounded-xl`).

## Color Palette (Tailwind 4.3 Directives)
- **Background (Main):** `#0A0E17`
- **Surface/Cards:** `#1E293B`
- **Structural Borders:** `#334155`
- **Primary/Performance Accent:** `#00C49F` (Emerald Green)
- **Primary Text:** `#F1F5F9`
- **Secondary/Muted Text:** `#94A3B8`
